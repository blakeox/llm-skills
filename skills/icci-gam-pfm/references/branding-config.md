# ICCI Branding Configuration

## Current Branding Repository

```
https://github.com/icci/icci-website-refresh-project.git
```

Local clone: `~/Documents/GitHub/icci-website-refresh-project/`

When generating reports, pull the latest branding from this repo. If the repo is unavailable, use the fallback values below.

## Switching Branding Repositories

When ICCI creates a dedicated theme repo, update this file:
1. Change the URL above to the new repo
2. Update the local clone path
3. Update any asset paths that change
4. Sync this file to GitHub
